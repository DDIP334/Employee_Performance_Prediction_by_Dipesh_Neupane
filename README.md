# Employee_Performance_Prediction_by_Dipesh_Neupane
IBM HR Employee Performance Prediction — KCE081BCT012


## Project Overview
This project predicts IBM employee Performance Rating (3=Good, 4=Excellent)
using Machine Learning on IBM HR Analytics dataset.

## Dataset
- Source: IBM HR Analytics Employee Attrition & Performance (Kaggle)
- Rows: 1,470 | Columns: 35
- Target: PerformanceRating

## Steps Completed
- Data Loading and Exploration
- Missing Value and Duplicate Check
- Outlier Detection and Handling (IQR Capping)
- Univariate Analysis (Histograms, Bar Plots, Countplots)
- Correlation Analysis (Heatmap)
- Assigned EDA (10 analyses)
- Bivariate Analysis (Pairplot)
- Feature Classification
- Label Encoding
- Log Transformation + StandardScaler
